# CV Tecnico - Thiago Aguiar ##

## Este es mi Currículum Vitae profesional, desarrollado como proyecto de aprendizaje

## Tecnologias usadas durante la creacion: 
* **HTML5:** Estructura semántica 
* **CSS3:** Diseño avanzado con **CSS Grid**
* **Efectos:** Interactividad con trasiciones y efectos de brillo (Glow)

## Características Principales
- **Diseño Responsivo:** Se adapta a móviles y PC sin necesidad de Media Queries complejas gracias a `grid-template-columns`.
- **UI Estilo Gamer:** Paleta de colores oscura con acentos naranja neón.
- **Optimización de Caja:** Uso de `border-box` para un diseño preciso.
